# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-25

### Added
- Initial public release of the Database Dev Stack.
- MySQL 8.0, PostgreSQL 13, Redis, Mailpit services.
- Web UIs: phpMyAdmin and Adminer.
- Persistent volumes and custom MySQL config/logs.
- Quick Start guide, connection examples (PHP/Node/Python), troubleshooting.
- CI workflow to validate docker-compose syntax.
- Release workflow to publish GitHub releases on tag push.
